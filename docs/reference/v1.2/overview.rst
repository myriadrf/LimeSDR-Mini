Overview
########

The heart of the LimeSDR Mini board is Intel MAX 10 (10M16SAU169C8G 169-UBGA) FPGA. It’s main function is to transfer digital data between the PC through a USB3.0 connector. The block diagram for LimeSDR-Mini board is presented in the Figure 3.

.. figure:: /images/LimeSDR-Mini_v1.2_block.png
  :width: 600

  Figure 3: LimeSDR Mini v1.2 board block diagram

This section contains component location description on the board. LimeSDR Mini v1.2 board picture with highlighted connectors and main components are presented in Figure 4 and Figure 5, respectively.

.. figure:: /images/LimeSDR-Mini_v1.2_top_componens.png
  :width: 600

  Figure 4: LimeSDR Mini v1.2 board top connectors and main components

.. figure:: /images/LimeSDR-Mini_v1.2_bot_componens.png
  :width: 600

  Figure 5: LimeSDR Mini v1.2 board bottom connectors and main components
